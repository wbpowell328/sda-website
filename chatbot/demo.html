CASTLE Chatbot — Demo Page

Local prototype. Click the chat bubble in the bottom-right corner to talk to Claude.

What this is

This page hosts the chatbot widget so you can test it end-to-end without touching the main Jekyll site. The widget code is in chatbot/widget.html and is included below.

Try asking

  • "What books has Warren written?"
  • "What is sequential decision analytics?"
  • "Where can I find his courses?"
  • "Who is Warren Powell?"

What's happening under the hood

Your browser sends a POST to http://localhost:3000/chat. The Node server (running on your laptop) attaches your knowledge base as a system prompt and forwards the conversation to the Claude API. The response streams back token by token. The Anthropic API key never leaves your laptop.
